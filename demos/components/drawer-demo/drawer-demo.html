Design & API Documentation

Selector: mdc-drawer

Properties Description
drawer: string Set the drawer implementation. (Default is 'permanent') 
Valid values: 'permanent' | 'persistent' | 'temporary'
fixed: boolean Makes permanent drawer fixed on top and have persistent elevation. 
* Permanent drawer only
fixedAdjustElement: any Reference to a drawer's attached body element. Default is document.body.
closeOnClick: boolean If false, clicking on the drawer will not close it. Default is true. 
* Temporary drawer only
Methods Description
open() Opens the drawer.
close() Closes the drawer.
isOpen(): boolean Returns whether or not the drawer is open.
isDrawerPermanent(): boolean Returns whether or not the drawer is permanent.
isDrawerTemporary(): boolean Returns whether or not the drawer is temporary.
isDrawerPersistent(): boolean Returns whether or not the drawer is persistent.
Events Description
opened() Event dispatched on drawer open.
closed() Event dispatched on drawer close.

Selector: mdc-drawer-spacer

Optional. Add to provide the matching amount of space for toolbar.

Selector: mdc-drawer-header

Optional. Add to create a 16:9 drawer header.

Selector: mdc-drawer-header-content

Optional. Use to add content inside mdc-drawer-header.
Properties Description
primary: boolean If true, sets mdc-theme--primary-bg and mdc-theme--text-primary-on-primary on mdc-drawer-header-content. Default is true.

Selector: mdc-drawer-content

Optional. Should be set on the list of items inside the drawer.
